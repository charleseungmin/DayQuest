## 1) 화면별 와이어프레임 명세

### A. Today
- 목적: 오늘의 핵심 진행 상태를 한 화면에서 빠르게 파악
- 블록
  - Header: `Today` + 서브카피(오늘 리듬 확인)
  - Summary Card: 완료/전체, 연속 기록, 다음 알림 시각
  - Checklist Card: 오늘 할 일 3개 내외의 축약 리스트
- 주요 상태
  - 기본 상태(요약 + 체크리스트)

### B. Manage
- 목적: Task 등록/수정/완료/삭제 작업 수행
- 블록
  - Header: `Manage`
  - State Switcher: 로딩/빈상태/오류 전환 점검
  - Progress Card: 완료 수/전체 수
  - Input Form: 제목/카테고리 + 추가/저장
  - Task List: 완료/수정/삭제 액션
  - Completion Banner: 전체 완료 시 축하 배너
- 주요 상태
  - Loading / Ready(empty|list) / Error

### C. History
- 목적: 일간/주간 진행 요약 확인
- 블록
  - Header: `History`
  - Today Summary Card
  - Weekly Summary Card

### D. Settings
- 목적: 알림/초기화 옵션 제어
- 블록
  - Header: `Settings`
  - Notification Toggle Card
  - Reset Card
